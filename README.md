# Writeups

Public write-ups from Hack The Box machines, CTF challenges, and (disclosure-approved) bug bounty reports.

This repo only contains material cleared for public disclosure — see the rule below. Private recon, in-progress bounty work, and credentials live elsewhere and are never pushed here.

## Structure

- `htb/` — retired Hack The Box machines (always safe to publish once retired)
- `ctf/` — CTF challenges
- `bugbounty/` — bug bounty reports, **only** after the program has closed the report and explicitly authorized public disclosure

## Format

Each write-up follows [`TEMPLATE.md`](TEMPLATE.md): target, vulnerability, attack vector, and what I'd do differently next time.
